## Соответствие требованиям брифа

Проект полностью соответствует изначальному техническому заданию и реализует полноценный AI-репетитор на основе современного RAG-подхода.

*   **Модульная архитектура:** Четкое разделение на бэкенд (`rest-api`), фронтенд (`AiRepetitor`) обеспечивает поддерживаемость и возможность независимого развития компонентов.
*   **Полноценный RAG-pipeline:** Реализация включает все ключевые этапы: индексацию учебных материалов, семантический поиск и интеллектуальную генерацию ответов на основе извлеченного контекста.
*   **Соблюдение технологического стека:** Использованы заявленные инструменты (FastAPI, LangChain, Docker).

## Соответствие задачам заказчика

*   **Полноценный RAG-pipeline:** Реализация включает ключевые этапы: индексацию учебных материалов, семантический поиск и интеллектуальную генерацию ответов на основе извлеченного контекста.
*   **Персонализация обучения:** Система авторизации и хранения истории диалогов обеспечивает индивидуальный подход к каждому пользователю.
*   **Контроль качества и этики:** Модуль тестирования позволяет объективно оценить усвоение материала, а фильтр нецензурной лексики поддерживает корректность образовательной среды.
*   **Адаптивная образовательная среда:** Интеграция всех модулей создаёт замкнутый цикл обучения: изучение → практика → проверка → корректировка.

## Логичность и удобство пользовательского интерфейса
Для открытия проекта и веб-нитерфейса требуется запуск из докера
```
docker compose up -d --build
```
Веб интерфес открывается по адресу: [localhost/8090](http://localhost:8090/)

Основной функционал после регистрации:
чат, открытие топиков, тестов, просмотр истории



## Масштабируемость, этика и качество документации
Логика реализованна на базе FastAPI что позволит подключиться другому разработчик и дополнить результат. Все основные логические модули вынесены в отдельные файлы в папки   [lib](rest-api/src/lib). Для реализации этики был добавлен отдельный класс [RussianSwearDetector](rest-api/src/lib/swear_detector.py) Которые выявляет матерные слова по средствам регулярных выражений и анализа контекста с помощью llm модели. Для удобства сборки добавлен docker-compose файл который разворачивает все сервисы и запускает окружения. Так же задокументированы все шаги для запуска проекта
